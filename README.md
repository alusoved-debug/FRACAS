# FRACAS Dashboard

דשבורד ניהול תקלות FRACAS (Failure Reporting, Analysis, and Corrective Action System).

## הפעלה ברשת פנימית (מומלץ)

האפליקציה יכולה להתחבר **אוטומטית** לקובץ Excel/CSV משותף בכל פתיחה, ולרענן אותו באופן מחזורי.

### התקנה בתיקייה משותפת

1. העתק לתיקייה ברשת (למשל `\\server\FRACAS\` או כונן ממופה):
   - `fracas-dashboard.html`
   - `fracas-config.json`
   - `AI_unclassified.xlsx` ← **קובץ הנתונים החי**
   - `start-fracas.bat`

2. עדכן במידת הצורך את `fracas-config.json`:

```json
{
  "dataUrl": "AI_unclassified.xlsx",
  "refreshHours": 6,
  "autoApply": true,
  "label": "קובץ נתונים משותף"
}
```

3. הפעל בלחיצה כפולה על **`start-fracas.bat`**  
   (דורש Python במחשב — מפעיל שרת מקומי קטן ומפתח את הדשבורד)

> **למה שרת מקומי?** דפדפן שפותח HTML כקובץ (`file://`) חוסם טעינה אוטומטית של Excel. השרת המקומי מאפשר חיבור קבוע לקובץ המשותף.

### איך זה עובד

| שלב | מה קורה |
|-----|---------|
| פתיחה | הדשבורד טוען אוטומטית את הקובץ מ-`dataUrl` |
| עדכון קובץ | כשמישהו מעדכן את ה-Excel בתיקייה — ברענון הבא הנתונים מתחלפים |
| רענון | כל 6 שעות (ברירת מחדל) בלי לטעון מחדש ידנית |
| כשל | אם הקובץ לא זמין — מוצגים נתוני הדמו המוטמעים |

בלשונית **העלאת נתונים** יש פאנל **"מקור נתונים קבוע"** עם סטטוס חיבור, כפתור רענון, ושמירת נתיב.

### חלופות ארגוניות

- **IIS / שרת קבצים פנימי**: פרסם את אותה תיקייה כאתר סטטי — אין צורך ב-`start-fracas.bat`
- **נתיב אחר**: שנה `dataUrl` ל-`data/FRACAS_live.xlsx` או לכתובת HTTP פנימית
- **פרמטר URL**: `fracas-dashboard.html?data=AI_unclassified.xlsx`

## הפעלה חד-פעמית (ללא חיבור חי)

פתח את `fracas-dashboard.html` ישירות בדפדפן — יוצגו נתוני הדמו. ניתן להעלות קובץ ידנית מלשונית ההעלאה.

## מה כלול

| קובץ | תיאור |
|------|--------|
| `fracas-dashboard.html` | האפליקציה המלאה (עצמאית, ללא CDN) |
| `fracas-config.json` | הגדרות מקור הנתונים הקבוע |
| `AI_unclassified.xlsx` | קובץ התקלות החי |
| `start-fracas.bat` | מפעיל שרת מקומי + פותח דפדפן |

## יכולות

- חיבור אוטומטי לקובץ Excel/CSV משותף + רענון מחזורי
- אשף ניתוח 10 שלבים (Ver_10): ולידציה, סוגי הודעה, כפילויות, Pareto, פעילות מתקנת, AS9100, RCA, CAPA, דוחות A–I (כולל NCR/8D), Executive Summary
- ייצוא Word / PDF לכל דוח
- גרפים, פילטרים, העלאה ידנית כגיבוי
- ממשק עברי RTL מלא

## טכנולוגיות

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js + SheetJS מוטמעים בקובץ (עובד ללא אינטרנט חיצוני)
